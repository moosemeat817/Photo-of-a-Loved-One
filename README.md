# Photo of a Loved One
This mod allows you to carry a customizable picture of something you care about.

<img src="https://github.com/moosemeat817/ModListJson/blob/main/photomod.png" width="80%">

## In your pocket, you find an old picture...

Inspired by a S.T.A.L.K.E.R mod of the same name. This mod allows you to carry a customizable picture of something you care about. Or something funny, your choice.

Inspect your picture using T, can be changed in the ModSettings menu.

⚠️ Make sure you keep the name of the photo file exactly as `Polaroid.png` ⚠️

---

- Crafted using two Wolf Pelts and two Cured Guts
- The item name is `GEAR_wolfscarf`
